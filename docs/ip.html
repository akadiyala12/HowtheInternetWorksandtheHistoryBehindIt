IP Addresses

What is a IP address?

An IP address (Internet Protocol address) is a numerical label assigned to each device connected to a computer network that communicates using the Internet Protocol. An IP address is used for two purposes: identifying a host or network interface, and addressing a specific location.

An IP address is defined as a 32-bit number in Internet Protocol version 4 (IPv4). However, due to the rapid growth of the Internet and the exhaustion of usable IPv4 addresses, a new version of IP (IPv6) was standardized in 1998, with a 128-bit IP address. Since the mid-2000s, IPv6 implementation has been ongoing.

How do they work?

An IP address has two primary functions: it defines the host, or more precisely its network interface, and it indicates the host's position in the network, allowing a path to be established to that host. Its role has been defined as follows: "A name indicates what we are looking for. Its location is indicated by an address. A route is a map that shows how to get there. Each IP packet's header contains the sending host's IP address as well as the destination host's.

IP Addresses are organized in a hierarchy. Traditional IP addresses are 32 bits long with 8 bits for each part of the address. The earlier numbers identify the country and original network of the device. Then the following numbers are identifying the subnetworks and the numbers after that are the address of the specific device. This version of IP addressing is called IPv4 and was designed in 1973 and widely adopted in the early 1980's. Overall it provides more than 4 billion unique addresses.

Those 4 billion addresses won't be enough in the long run so we are now in transition to a new form of IP addressing called IPv6. This new address is 128 bits per address and can now support 240 undecillion unique addresses.
