Domain Name System

What is it?

The Domain Name System (DNS) is the Internet's address book. Domain names such as nytimes.com and espn.com allow people to access information on the internet. Internet Protocol (IP) addresses are used to communicate between web browsers. DNS converts domain names to IP addresses, allowing browsers to access services on the Internet.

Each Internet-connected computer has a unique IP address that other machines use to locate it. DNS servers remove the need for humans to memorize IP addresses like 192.168.1.1 (in IPv4) or more complex newer alphanumeric IP addresses like 2400:cb00:2048:1::c629:d7a2 (in IPv6).

How does DNS work?

DNS resolution is the method of transforming a hostname (for example, www.github.com) into a computer-friendly IP address (such as 192.168.1.1). - device on the Internet is assigned an IP address, which is required to locate the appropriate Internet device, much like a street address is required to locate a specific home.

When a user requests a webpage, a conversion must take place between the user's input (example.com) and the machine-friendly address required to find the example.com webpage.
